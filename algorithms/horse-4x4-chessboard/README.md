# 马与4×4棋盘

**类型：** 马尔可夫链 · 期望值计算
**难度：** ⭐⭐⭐
**答案：** 期望 **6步**

---

## 问题描述

在一个 4×4 的棋盘上，有一匹马遵循象棋中"马走日"的规则行动：每次可以移动到距当前位置水平±1垂直±2，或水平±2垂直±1的格子（须在棋盘内）。

棋盘的四个角落是：`(0,0)`, `(0,3)`, `(3,0)`, `(3,3)`

**问：马从任意一个角落出发，再次踏上任意角落（包括原角落）所需步数的期望值是多少？**

```
  0   1   2   3
0 [★] [ ] [ ] [★]
1 [ ] [ ] [ ] [ ]
2 [ ] [ ] [ ] [ ]
3 [★] [ ] [ ] [★]

★ = 角落（目标格子）
```

---

## 直觉分析

首先，我们来感受一下马的运动有多受限：

从角落 `(0,0)` 出发，马**只有 2 个**合法落点：`(1,2)` 和 `(2,1)`。
这两个格子都是内部格，而从内部格出发，每一步有机会回到角落。

关键观察：
- **角落格**：最少合法走法（只有 2 步），且直接通向内部格
- **内部格** `(1,1)`, `(1,2)`, `(2,1)`, `(2,2)`：每步恰好有 **4 个**合法走法，其中 2 个是角落
- **边缘非角格**：3 个合法走法，无法直接到达角落

---

## 数学建模：马尔可夫链

定义随机变量 $E(r,c)$ = 从格子 $(r,c)$ 出发，**首次**到达任意角落所需的期望步数。

**边界条件：** 角落格的期望步数为 0（已在目标位置）：
$$E(\text{角落}) = 0$$

**递推关系：** 对于非角落格 $(r,c)$，设其合法邻居集合为 $N(r,c)$：
$$E(r,c) = 1 + \frac{1}{|N(r,c)|} \sum_{(r',c') \in N(r,c)} E(r',c')$$

含义：走一步（+1），然后从新位置继续。

---

## 求解：利用对称性

4×4 棋盘具有 D4 对称性（旋转+翻转），16 个格子恰好分为 **3 种等价类**：

```
  0    1    2    3
0 [C] [B] [B] [C]     C = 角落 (4个)  → E_C = 0
1 [B] [I] [I] [B]     B = 边非角落 (8个) → E_B = ?
2 [B] [I] [I] [B]     I = 内部格 (4个)   → E_I = ?
3 [C] [B] [B] [C]
```

### 建立方程

**B 类代表格 `(0,1)`** 的合法走法：

```
(0,1) → (1,3)[B]、(2,2)[I]、(2,0)[B]    共3步，2个B + 1个I
```

$$E_B = 1 + \frac{2}{3}E_B + \frac{1}{3}E_I \quad \cdots (1)$$

**I 类代表格 `(1,1)`** 的合法走法：

```
(1,1) → (2,3)[B]、(0,3)[C]、(3,2)[B]、(3,0)[C]    共4步，2个B + 2个C
```

$$E_I = 1 + \frac{2}{4}E_B + \frac{2}{4} \cdot 0 = 1 + \frac{1}{2}E_B \quad \cdots (2)$$

### 求解方程组

将 $(2)$ 代入 $(1)$：

$$E_B = 1 + \frac{2}{3}E_B + \frac{1}{3}\left(1 + \frac{1}{2}E_B\right)$$

$$E_B = \frac{4}{3} + \frac{2}{3}E_B + \frac{1}{6}E_B = \frac{4}{3} + \frac{5}{6}E_B$$

$$E_B\left(1 - \frac{5}{6}\right) = \frac{4}{3} \implies \frac{1}{6}E_B = \frac{4}{3}$$

$$\boxed{E_B = 8, \quad E_I = 5}$$

### 计算从角落出发的期望步数

从角落 `(0,0)` 出发，仅有 2 个合法走法：`(1,2)[I]` 和 `(2,1)[I]`，均为 **I 类**：

$$E_{\text{从角落出发}} = 1 + E_I = 1 + 5 = \boxed{6}$$

---

## 所有格子的期望步数

```
  0    1    2    3
0 [C] [ 8] [ 8] [C]
1 [ 8] [ 5] [ 5] [ 8]
2 [ 8] [ 5] [ 5] [ 8]
3 [C] [ 8] [ 8] [C]

C = 0（目标）
```

> 规律：内部格距角落最近（5步），边缘格更远（8步），对称优美。

def monte_carlo(n_trials=1_000_000):
    total = 0
    for _ in range(n_trials):
        r, c = 0, 0   # 从角落出发
        steps = 0
        while True:
            r, c = random.choice(valid_moves(r, c))
            steps += 1
            if (r, c) in CORNERS:
                break
        total += steps
    return total / n_trials  # ≈ 6.00

## 总结

| 方法 | 时间复杂度 | 核心思路 |
|------|-----------|---------|
| 对称性推导 | O(1) | 3 类等价格 → 2元方程组 |
| 线性方程组 | O(n³) | 12个未知数高斯消元 |
| Monte Carlo | O(k) | 百万次随机游走取均值 |

**答案：从角落出发，期望经过 6 步再次到达角落。**

这道题优雅之处在于：尽管棋盘只有 16 格，马的走法限制极多，利用 D4 对称性将问题化简为仅需解 2 个方程的线性系统。
